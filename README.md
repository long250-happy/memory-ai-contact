## 一、核心读心机制

### 1.1 基本规则
- **爱到一定程度才能读心**：对某角色的感情/羁绊达到一定阈值后，方可读取其内心。
- **无重复读心**：读心的内容无法被其他人再次读取。即：不能读别人读心的内容，也不能读因此读心产生的心理活动。

- ### 5.1 读心在幻想世界中的定位
- **读心并非常识**：该世界中，读心不是人人皆知的能力。

- 因此，角色之间可能互相不知道对方也会读心，形成隐藏的信息差博弈。
